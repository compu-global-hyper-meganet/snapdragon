# snapdragon

## Run the client - dev

1. `cd client\ui`
2. `npm start`

## Build the client

1. `cd client\ui`
2. `npm run build`

This will output into `client\ui\dist`

## Run the server

1. `cd server`
2. `npm start`

This will serve the build client from `client\ui\dist` so make sure you build the client first
